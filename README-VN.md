# Ứng dụng Quản lý Siêu thị Mini trên Desktop

Ứng dụng desktop theo vai trò để quản lý siêu thị mini.  
Dự án này được phát triển như một bài tập học thuật nhằm thực hành **Fullstack Development** với **C#**, **WinForms**, và **Microsoft SQL Server**.

---

## 📌 Thông tin Dự án
- **Quy mô nhóm:** 2  
- **Vai trò:** Fullstack Developer  
- **Thời gian:** Tháng 9/2025 – Hiện tại  
- **Ngôn ngữ/Frameworks:** C#, WinForms, .NET 9.0 & .NET Framework 4.8  

---

## 📌 Tổng quan
Hệ thống được thiết kế để quản lý các hoạt động của siêu thị bao gồm tồn kho sản phẩm, đơn hàng, nhà cung cấp và khách hàng.  
Ứng dụng triển khai **Kiến trúc 3 lớp** và hỗ trợ kiểm soát truy cập theo vai trò.

---

## 📌 Tính năng
- <mark>Xác thực</mark> (Đăng nhập/Đăng ký)  
- <mark>Kiểm soát truy cập theo vai trò</mark> (Admin, Nhân viên)  
- <mark>Quản lý sản phẩm/đơn hàng</mark>  
- <mark>Quản lý nhà cung cấp/khách hàng</mark>  
- <mark>Thống kê</mark> và <mark>Lịch sử hệ thống</mark>  
- <mark>Khôi phục mật khẩu qua Email</mark>  
- <mark>Chuyển đổi chế độ Sáng/Tối</mark>  

---

## 📌 Thiết kế Hệ thống
- Thiết kế <mark>ERD</mark> cho sơ đồ cơ sở dữ liệu  
- Triển khai <mark>Kiến trúc 3 lớp</mark> để dễ bảo trì  
- Cơ sở dữ liệu xây dựng trên <mark>Microsoft SQL Server</mark>  

---

## 📌 Cách chạy
1. Clone repository:
   ```bash
   git clone https://github.com/DuyKhoiCoder30062004/C--Mini-Market.git
2. Mở dự án trong Visual Studio.
3. Cấu hình chuỗi kết nối SQL Server trong file app.config hoặc Program.cs.
4. Build và chạy solution.
